Action button Description
Create new user

You can create new user with specific role. You can create:

  1. Customer
  2. Restaurant owner
  3. Delivery man
  4. Rest api user

Services by role
Customer

Restaurant Customer panel. To gain access you have to log in.
You can log in as:

  • Previous created user
  • Default user with:
    • Username: default_customer
    • Email: customerEmail@mail.com
    • Password: top_secret123
